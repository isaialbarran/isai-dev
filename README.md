# isai-dev

Personal portfolio — [isaialbarran.com](https://www.isaialbarran.com).

Plain HTML + CSS. No build step. Hosted via GitHub Pages or Vercel on
`www.isaialbarran.com`.

## Structure

- `index.html` — the whole site (hero, about, experience, projects, stack, contact)
- `style.css` — dark theme, responsive
- `assets/Isai_Albarran_CV.pdf` — downloadable CV
- `isai-icon.jpg` — favicon / OG image
- `CNAME` — custom domain for GitHub Pages
- `robots.txt`, `sitemap.xml` — SEO

## Local preview

Open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8080
```

Then visit <http://localhost:8080>.
